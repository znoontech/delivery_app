POS Delivery Mode Selction

Allow the cashier to select the type of the Order such as 'pickup' or 'delivery'

The module the cashier to select the Delivery mode options namely 'pickup' and 'delivery'. These order types are save in the backend along with order. You can enable customer with 'Home Delivery'. Upon selection of the 'Home Delivery' enabled customer , the system will select the 'Home delivery' type automatically. Once 'Home Delivery' is selected, it also automaticall enable the 'Customer Credit' Journal in the payment screen.

  • 1. Delivery mode options.
  • 2. Customer based delivery option.
  • 3. Print delivery option in the receipt.
  • 3. Customer/Delivery mode based enabling credit journal.

Usage

  • Enable 'Home Delivery' in pos config

  • Create Customer with 'Home Delivery'

  • Open POS session
  • Create an order
  • Select the customer with 'Home delivery' ticked

  • Payment screen

  • Receipt Screen

  • Order Screen
